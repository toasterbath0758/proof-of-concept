## Beschrijving
<!-- In de Beschrijving staat kort beschreven wat voor project het is en wat je hebt gemaakt -->
De vraag: maak een app zodat de mensen die geinteresseerd zijn om te interageren met de bloemen in de tuin.
volg hierbij het bestaande prototype. (veranderingen mogelijk met overleg).

De opdracht die ik op mezelf heb genomen is de nieuwspagina en een nieuwe interactie, namelijk de suggestions page.
hier kan je een suggestie achterlaten, je voert in wat je favoriete plant is en waarom je deze in de bloementuin zou willen zien.

3. Open dan de terminal (ctrl + j op windows) en type npm install
4. Start vervolgens de website door npm start in te typen
